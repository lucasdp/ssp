# Simple Seed Project

## for frontend developers

*Gulp, SASS, Bootstrap, animate.css*

*Image optimizer and File minifier*

### Installation & Usage

`$ git clone`

`$ npm install`

`$ gulp`

### External Dependencies

- libjpeg
- libpng

`brew install libjpeg libpng` on macOS
`apt-get install -y libjpeg libpng` on Ubuntu
